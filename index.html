BlackBerry QNX
Software Testing Student - Core OS Test (May 2019 - August 2019)

BlackBerry is a software company that specializes in enterprise security, offering solutions such as the world's first endpoint security system to apply AI and Machine Learning for virus and malware detection. My second internship position was also at BlackBerry's QNX division, which develops a secure Unix-like RTOS and virtual machine for safety-critical applications such as driverless vehicles, aviation, and medical devices.

As part of the Core OS Test team, my job was to create automated stress tests for QNX's Hypervisor (virtual machine) that were compatible with multiple hardware platforms. These tests used to be performed manually. Additionally, I ran performance benchmarks on new Hypervisor builds and documented performance comparisons against older builds to find potential areas of defficiency. Overall, it was a great learning experience.
